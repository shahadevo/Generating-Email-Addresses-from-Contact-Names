# Generating-Email-Addresses-from-Contact-Names


## Overview

This project demonstrates how to clean and transform customer contact data using SQL. The query extracts the first and last names from the `primary_poc` field and generates standardized email addresses based on customer names and company information.

## Objectives

* Extract first and last names from contact information.
* Perform data cleaning using SQL string functions.
* Generate consistent email addresses.
* Improve data quality and standardization.

## SQL Functions Used

* `LEFT()`
* `RIGHT()`
* `STRPOS()`
* `LENGTH()`
* `CONCAT()`

## Output

The query produces:

* First Name
* Last Name
* Generated Email Address
